# NeoBiz Solutions 웹사이트

간단한 비즈니스 컨설팅 서비스를 소개하는 정적 웹사이트입니다. 히어로, 서비스, 솔루션 프로세스, 성과 사례, 상담 문의 섹션으로 구성되어 있습니다.

## 파일 구조

- `index.html` – 메인 페이지 마크업
- `styles.css` – 레이아웃과 테마 스타일
- `script.js` – 모바일 내비게이션 토글 등의 인터랙션

## 로컬 미리보기

```bash
python3 -m http.server 8000
```

위 명령을 실행한 후 브라우저에서 <http://localhost:8000>을 열어 사이트를 확인할 수 있습니다.
